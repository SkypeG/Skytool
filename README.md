# Skytool

## Tested On

- Windows 10
- Ubuntu
- Kali Linux

## Run In Windows
Clone the project

```bash
  git clone https://github.com/SkypeG/Skytool
```

Install Library Skytool

```bash
  Run @Install.bat
```

Run Skytool

```bash
  Run @Run.bat
```

## Run In Linux
Clone the project

```bash
  git clone https://github.com/SkypeG/Skytool
```

Go to the project directory

```bash
  cd Skytool
```

Run Skytool

```bash
  python3 Skytool.py
```
